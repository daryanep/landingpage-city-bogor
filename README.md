# 🌿 Bogor City Landing Page

A simple and responsive landing page showcasing the beauty, culture, and attractions of Bogor City, Indonesia. This project is built using only **HTML5** and **CSS3**, making it lightweight, fast, and easy to customize.

## Live Preview

[Click Here](https://daryanep.github.io/landingpage-city-bogor/)

## 📌 Features

* Responsive design
* Modern and clean user interface
* Hero section with city introduction
* About Bogor section
* Smooth scrolling navigation
* Pure HTML & CSS (No JavaScript)

## 🛠️ Technologies Used

* HTML5
* CSS3

## 🚀 Getting Started

1. Clone this repository:

```bash
git clone https://github.com/yourusername/landingpage-city-bogor.git
```

2. Navigate to the project folder:

```bash
cd landingpage-city-bogor
```

3. Open `index.html` in your browser.

No additional installation or dependencies are required.

## 🌄 About Bogor

Bogor, known as the "Rain City" of Indonesia, is famous for its cool climate, lush greenery, historical landmarks, and the iconic Bogor Botanical Gardens. This landing page aims to introduce visitors to the city's charm through a modern web design.
